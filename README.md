# טיימר אליאס

טיימר דיגיטלי למשחק **אליאס** — המשחק שבו שחקן אחד מסביר מילים לקבוצה שלו בלי להשתמש במילה עצמה, בזמן מוגבל.

## תכונות

- ⏱️ שני מצבים: תור רגיל (תור הסבר → זמן ניחושים) ותור מיוחד (עד 5 מילים ברצף)
- ⚙️ הגדרות זמן מותאמות אישית (זמן תור, זמן ניחושים, זמן תור מיוחד)
- 💾 שמירה אוטומטית של ההגדרות בדפדפן
- ⌨️ קיצורי מקלדת: רווח (התחל/השהה), R (איפוס), N (למילה הבאה)
- 📱 ניתן להתקנה כאפליקציה (PWA) ולעבוד גם אופליין
- 🌐 ממשק מלא בעברית ו-RTL
- 🔔 צלילי אזהרה ב-5 שניות האחרונות ובוזר בסיום הזמן

## איך משחקים

1. בחרו מצב: **תור רגיל** (לסיבוב מילים רגיל) או **תור מיוחד** (למשחק מילים מהיר)
2. לחצו על הכפתור הגדול כדי להתחיל — יש לכם `60 שניות` (או כמה שהגדרתם) להסביר
3. כשנגמר הזמן — עוברים אוטומטית ל-`30 שניות` של ניחושים
4. בסוף הניחושים — לחצו **איפוס** כדי להתחיל סיבוב חדש
5. בתור מיוחד — לחצו **למילה הבאה** כדי לעבור למילה הבאה (עד 5 מילים)

### קיצורי מקלדת

- `רווח` — התחל / השהה את הטיימר
- `R` — איפוס
- `N` — למילה הבאה (רק בתור מיוחד)

## התאמה אישית

לחצו על סמל ⚙️ (הגדרות) כדי לשנות:
- **זמן תור** (10-300 שניות, ברירת מחדל: 60)
- **זמן ניחושים** (5-120 שניות, ברירת מחדל: 30)
- **זמן תור מיוחד** (10-300 שניות, ברירת מחדל: 45)

ההגדרות נשמרות אוטומטית בדפדפן.

## התקנה כאפליקציה

האפליקציה תומכת ב-PWA. בדפדפנים נתמכים (Chrome, Edge, Safari), לחצו על סמל **התקן** שיופיע בפינה הימנית-עליונה כדי להוסיף את הטיימר למסך הבית. אחרי ההתקנה האפליקציה עובדת גם בלי אינטרנט.

## פיתוח מקומי

```bash
# התקנת תלויות
npm install

# הרצת שרת פיתוח
npm run dev
# → http://localhost:8080

# בניית גרסת ייצור
npm run build

# תצוגה מקדימה של גרסת הייצור
npm run preview
```

## פריסה (Deploy)

האפליקציה מוכנה לפריסה ב-[Coolify](https://coolify.io) באמצעות **Nixpacks** (ברירת המחדל של Coolify — אין צורך ב-Dockerfile):

1. חברו את ה-repo (`https://github.com/uri879/Finkalias`) למופע Coolify שלכם
2. Coolify יזהה אוטומטית שמדובר ב-Vite (לפי `vite.config.ts`)
3. פקודת בנייה: `npm run build`
4. תיקיית פלט: `dist/`
5. Coolify יגיש את `dist/` כאתר סטטי

אין צורך ב-Dockerfile, nginx.conf או .dockerignore.
